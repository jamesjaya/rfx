# rfx

rfx is a cli currency converter

## Installation

Use [cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html) to install rfx for now.

```bash
curl https://sh.rustup.rs -sSf | sh
```

## Usage

```bash
rfx <source> <target>
```

```bash
rfx USD IDR
```
